# BDevs - Web Hosting Services | HTML & CSS Responsive Web Page

A clean, responsive webpage built using only HTML and CSS, showcasing a modern layout and smooth design for the BDevs Hosting concept.

This project demonstrates how to fetch and render multiple components dynamically into a single HTML page using JavaScript, mimicking the behavior of React components. The page is fully responsive and utilizes HTML and CSS media queries to ensure a seamless experience across different screen sizes.

### Features

* Dynamic Component Rendering: Uses JavaScript to fetch and render components into the main page.

* Responsive Design: The layout adapts to different devices, ensuring a great user experience on desktop, tablet, and mobile devices.

### Languages used

* HTML: The structure of the web page.

* CSS: Used for styling and media queries to make the design responsive.

* JavaScript: To fetch and dynamically insert content into the page, similar to how React manages components.

### Installation

1. Clone this repository to your local machine

```git clone https://github.com/awizp/bdevs-landing-page.git
```

2. Open the ```index.html``` file in your browser to view the project.
